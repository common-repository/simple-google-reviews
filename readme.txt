A simple plugin to display your Google places reviews on your WordPress website.

== Installation ==
1. Install the Simple Google Reviews plugin either via the WordPress.org plugin directory, or by uploading the files to your server
2. After activating the Simple Google Reviews plugin, navigate to Settings -> Simple Google Reviews
3. Enter the appropriate settings and click save.
4. Navigate to the section of the website where you would like the reviews to appear and enter the shortcode <pre><code>[simplegooglereviews]</code></pre>

== Frequently Asked Questions ==
= What is my Google API key? =

You can create a Google API key [here](https://developers.google.com/maps/documentation/javascript/get-api-key).

= What is my Google Place ID? =

You may find your Google Place ID [here](https://developers.google.com/places/place-id).

= How do I install the plugin? =

Enter the shortcode <pre><code>[simplegooglereviews]</code></pre>

= Can I display more than 5 reviews? =

No, this restriction is set by Google.

== Screenshots ==

1. If you would like the reviews to rotate, select the number of seconds you would like each review to appear in the settings.
2. To disable the rotating reviews and display all of the reviews at once, select "No" in the option "Would you like to rotate the reviews?".
3. The settings screen will allow you to customize the options available.
